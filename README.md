# CH4EnemyRecreations-Kristal
A library for Kristal that recreates Chapter 4's enemies (bosses such as Jackenstein, Hammer/Sound of Justice &amp; the Titan will not be included for obvious reasons.)

# Enemies Included
* Guei
* Balthizard
* Bibliox
* Mizzle
* Winglade
* Organikk
* Wicabel
* Titan Spawn

# Other Features
* Additional animations for the Fun Gang, used in several enemy ACTs.
* A couple unused enemy attack recreations.
* Ports of `scr_lerpvar()`, `lengthdir_x/y()` and a few lerping functions from Deltarune.
